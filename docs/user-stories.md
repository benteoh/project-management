# User Stories

## Programme

| Priority | Story |
|----------|-------|
| High | As a PM, I can create a programme for a project |
| High | As a PM, I can breakdown the programme into scopes, tasks and activities |
| High | As a PM, I can specify the start and end dates for a work item |
| High | As a PM, I want to assign an original duration split across multiple engineers on a scope |
| High | As a PM, I want to be able to mark the progress of individual activities or tasks |
| Medium | As a PM, I want to see an overview of the amount of work required and the duration |
| Medium | As a PM, I want to be able to sort by finish date, filter by activity status and warning status |
| Medium | As a PM, I want to be able to export a programme as a csv or excel sheet |
| Low | As a PM, I want to be able to view the work items as a gantt chart and edit directly |
| Low | As a PM, I want to see forecasted hours per scope (broken down by person) |

## Demand Forecast

| Priority | Story |
|----------|-------|
| High | As a PM, I want to be able to add planned hours to engineers across their tasks per day |
| High | As a PM, for each engineer in a task, I want to be able to see their planned hours and demanded hours |
| High | As a PM, I want to have indication and restriction of only setting at most the max number of hours per day |
| Medium | As a PM, I want to be able to filter by task and people |
| Medium | As a PM, I want a focused view of relevant dates |
| Medium | As a PM, I want to be able to see engineer's hourly rates and cost based on demanded hours |
| Medium | As a PM, I want to be able to block out weekends, bank holidays and leave |
| Medium | As a PM, I want to be able to allocate hours for an engineer not involved in a project for extra cover |
| Medium | As a PM, I want an indication of the start and end date of the scope |
| Medium | As a PM, I want to be able to export the forecast as a csv or excel |
| Medium | As a PM, I can change a forecast due to sick leave |
